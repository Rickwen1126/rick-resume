串流與影像處理核心

負責 LiveView 多路串流即時切換系統，讓同一攝影機的多路串流能即時接入並切換播放。從 FFmpeg 原始碼移植 H.265 codec 支援。開發 NVR 專用 CLI 轉碼工具。實作 CGI 到 WebSocket 的協定串接，將傳統 request-response 介面升級為即時雙向通訊。

關鍵問題排除

多路影像同時播放時效能劣化近一倍，團隊長期無法定位原因。追蹤發現 root cause 是 frame 在 pipeline 中被 deep copy 導致 disk hit 飆高，修復後效能恢復正常並解決伴隨的 memory leak。另一個阻塞整體進度的問題是 Mac 平台 UI 移植，最終以 Qt fusion mode 一行修復跨平台相容性，解除團隊的進度瓶頸。

AI 協作開發

核心觀念：Preview > Review — 在 AI 時代，開發前的深度設計與架構選型，比事後逐行 code review 更有價值。

開發工具鏈

以 Claude Code CLI 作為主要開發環境，搭配自建的 MCP Server 生態系（NotebookLM Controller 用於查詢技術文獻降低幻覺、Skill 系統管理可復用的開發流程）。使用 Copilot SDK 建構 Line Bot Agent，實踐 AI-native 的應用架構。所有近期專案皆在此工具鏈下完成。

開發流程

每個功能先與 AI 深度討論需求、搜尋開源方案、產出完整 spec 後才動手實作。開發中分 phase 迭代，每段完成即以 CodeTour 標記重點做 review，聚焦界面、架構方向與安全性，控制在 10 分鐘內。學習上以 AI 拆解開源專案架構 — 分為架構層、邊界 / interface 層、功能層三層理解，快速建立新領域的技術視野。
